# TaskFlow — To-Do / Task Manager App

A sleek, production-quality **To-Do & Task Manager** app built with pure **HTML, CSS & JavaScript** — no frameworks, no dependencies.

---

## 🚀 Live Preview

Open `taskflow.html` directly in any browser — no build step required.

---

## ✨ Features

| Feature | Details |
|---|---|
| **Add tasks** | Title, category, priority, due date |
| **Complete tasks** | Animated checkbox with purple glow |
| **Edit tasks** | Full modal editor |
| **Delete tasks** | Smooth slide-out animation |
| **Filter & Search** | By status, priority, category, or keyword |
| **Progress ring** | Real-time circular progress indicator |
| **Confetti** | Fires when all tasks are completed 🎉 |
| **Dark / Light mode** | Persisted toggle with system preference fallback |
| **Categories** | Work, Personal, Shopping, Health, Learning |
| **Priority levels** | High / Medium / Low with color indicators |
| **Toast notifications** | Non-blocking status messages |
| **Animated stats** | Numbers count up on change |
| **Keyboard shortcuts** | `Enter` to add, `Esc` to close modal |
| **Responsive** | Full mobile support (375px+) |
| **Accessible** | Semantic HTML, ARIA labels, focus management, keyboard navigation |

---

## 🎨 Design System

- **Colors:** Black (`#0c0c10`), White (`#e8e6f0`), Purple (`#8b5cf6`) accent
- **Typography:** Cabinet Grotesk (display) + Inter (body)
- **Motion:** CSS cubic-bezier easing, confetti particle system, task enter/exit animations
- **Dark mode:** Default dark theme with one-click light mode toggle
